# Fishtech Hackathon

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Open data repository. Columns: name_ja, 代表的な魚介, 漁業の特徴, name_en.

## Demo

https://code4fukui.github.io/fishtech-hackathon/

## Features

- ES module compatible with Deno and modern browsers
- Open data CSV with columns: name_ja, 代表的な魚介, 漁業の特徴, name_en

## Data

Columns: name_ja, 代表的な魚介, 漁業の特徴, name_en, feature_type, latitude_deg

## Installation & Setup

No build step required. Clone and open in a browser or use with Deno:

```bash
git clone https://github.com/code4fukui/fishtech-hackathon.git
```
